# Key Bindings Reference

Blu is optimized for a keyboard-driven workflow. Below is a list of the primary keyboard shortcuts.

## Global Navigation

- `Ctrl+C`: Quit the application.
- `Ctrl+?`: Toggle the help overlay.
- `Ctrl+L`: Clear the current screen or view logs.
- `Ctrl+S`: Switch between different chat sessions.
- `Ctrl+K`: Open the command palette.
- `Ctrl+O`: Open the model selection dialog.
- `Ctrl+P`: Open the provider and API key configuration dialog.
- `Esc`: Close the current overlay or return to the previous view.

## Chat Interface

- `i`: Focus the message input field.
- `Esc`: Unfocus the input field to navigate message history.
- `Enter` or `Ctrl+S`: Send the current message.
- `Ctrl+X`: Stop the current AI generation or tool execution.
- `Ctrl+E`: Open the current input buffer in your external `$EDITOR`.

## Message and Tool Result Navigation

- `Up` / `Down`: Scroll through the conversation history.
- `PageUp` / `PageDown`: Scroll faster through history.

## Approval Dialogs

- `a`: Allow the proposed action (file edit or command execution).
- `A`: Allow the action and all subsequent similar actions for this session.
- `d`: Deny the proposed action.
